## 1. v2.logistics.get_mass_shipping_parameter

**Method:** `POST`  
**Path:** `/api/v2/logistics/get_mass_shipping_parameter`

### API Description
Use this api to check if package support pickup, dropoff, non-integrated. For pickup, return address and pickup time id options. For dropoff, return branch id, sender real name, etc. Can batch request for packages under same `product_location_id` and `logistics_channel_id`.

> Please call it when packages meet:
> 1. fulfillment status is `LOGISTICS_READY`; or
> 2. fulfillment status is `LOGISTICS_PICKUP_RETRY`; or
> 3. fulfillment status is `LOGISTICS_REQUEST_CREATED` and meet Instant Order Reschedule conditions

## 2. v2.logistics.mass_ship_order

**Method:** `POST`  
**Path:** `/api/v2/logistics/mass_ship_order`

### API Description

Use this api to initiate logistics including arrange pickup, dropoff or shipment for non-integrated logistic channels. Should call `v2.logistics.get_mass_shipping_parameter` to fetch all required params first. It's recommended to initiate logistics one hour after the orders were placed since there is one-hour window buyer can cancel any order without request to seller. The API can only batch arrange shipment for multiple packages under the same `product_location_id` and same `logistics_channel_id`.

## 3. v2.logistics.get_mass_tracking_number

**Method:** `POST`  
**Path:** `/api/v2/logistics/get_mass_tracking_number`

### API Description

After arranging shipment (`v2.logistics.mass_ship_order`) for the integrated channel, use this api to get the `tracking_number`, which is a required parameter for creating shipping labels. The api response can return `tracking_number` empty, since this info is dependent from the 3PL. Due to this, it is allowed to keep calling the api within 5 minutes interval, until the `tracking_number` is returned.

| Live | Brazil | `https://openplatform.shopee.com.br/api/v2/logistics/get_mass_tracking_number` |
| Sandbox | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/logistics/get_mass_tracking_number` |
| Sandbox | Chinese Mainland | `https://openplatform.sandbox.test-stable.shopee.cn/api/v2/logistics/get_mass_tracking_number` |
