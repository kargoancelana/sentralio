# Shopee API: v2.payment.get_escrow_detail

## API Description

Use this API to fetch the accounting detail of an order.

## Request

### Endpoint
- **Method**: GET
- **Path**: `/api/v2/payment/get_escrow_detail`

### Environment URLs

| Environment | Location | HTTP Address |
|-------------|----------|--------------|
| Live | Global | `https://partner.shopeemobile.com/api/v2/payment/get_escrow_detail` |
| Live | Chinese Mainland | `https://openplatform.shopee.cn/api/v2/payment/get_escrow_detail` |
| Live | Brazil | `https://openplatform.shopee.com.br/api/v2/payment/get_escrow_detail` |
| Sandbox | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/payment/get_escrow_detail` |
| Sandbox | Chinese Mainland | `https://openplatform.sandbox.test-stable.shopee.cn/api/v2/payment/get_escrow_detail` |

### Common Request Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| partner_id | int | 1 | Partner ID assigned upon registration. Required for all requests. |
| timestamp | timestamp | 1610000000 | Timestamp of the request. Required for all requests. Expires in 5 minutes. |
| access_token | string | c09222e3fc40ffb25fc947f738b1abf1 | Token for API access. Valid for multiple use, expires in 4 hours. |
| shop_id | int | 600000 | Shopee's unique identifier for a shop. Required for most APIs. |
| sign | string | e318d3e932719916a9f9ebb57e2011961bd47abfa54a36e040d050d8931596e2 | Signature generated via HMAC-SHA256. |

### Request Parameters

| Name | Type | Required | Sample | Description |
|------|------|----------|--------|-------------|
| order_sn | string | True | 220914R9U7D3C6 | Shopee's unique identifier for an order. |

### Request Example

```json

    "order_sn": "251021P1YSSN5U"
}
```
