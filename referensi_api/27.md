# Shopee API: v2.payment.get_income_statement

## API Description

To query income statement status and provide file link if the income statement is ready to be downloaded.

## Request

### Endpoint
- **Method**: GET
- **Path**: `/api/v2/payment/get_income_statement`

### Environment URLs

| Environment | Location | HTTP Address |
|-------------|----------|--------------|
| Live | Global | `https://partner.shopeemobile.com/api/v2/payment/get_income_statement` |
| Live | Chinese Mainland | `https://openplatform.shopee.cn/api/v2/payment/get_income_statement` |
| Live | Brazil | `https://openplatform.shopee.com.br/api/v2/payment/get_income_statement` |
| Sandbox | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/payment/get_income_statement` |
| Sandbox | Chinese Mainland | `https://openplatform.sandbox.test-stable.shopee.cn/api/v2/payment/get_income_statement` |

### Common Request Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| partner_id | int | 1 | Partner ID assigned upon registration. Required for all requests. |
| timestamp | timestamp | 1610000000 | Timestamp of the request. Required for all requests. Expires in 5 minutes. |
| access_token | string | c09222e3fc40ffb25fc947f738b1abf1 | Token for API access. Valid for multiple use, expires in 4 hours. |
| shop_id | int | 600000 | Shopee's unique identifier for a shop. Required for most APIs. |
| sign | string | e318d3e932719916a9f9ebb57e2011961bd47abfa54a36e040d050d8931596e2 | Signature generated via HMAC-SHA256. |

### Request Parameters

| Name | Type | Required | Sample | Description |
|------|------|----------|--------|-------------|
| income_statement_id | int64 | True | 123456 | The identifier for income statement file request. Returned from `v2.payment.generate_income_statement`. |

### Request Example

```
GET /api/v2/payment/get_income_statement?access_token=access_token&income_statement_id=123456&partner_id=partner_id&shop_id=123456&sign=sign&timestamp=timestamp
```

## Response

### Response Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| response | object | - | Response content. |
| response.id | int64 | 123456 | The identifier for income statement file request. |
| response.file_name | string | weekly_report_20241111.pdf | Income statement file name. |
| response.status | int32 | 1 | Status of the income statement. See status values below. |
| response.generated_time | int64 | 12345678987654 | File generation time. |
| response.file_link | string | https://... | Link to download income statement file. |
| error | string | - | Error code. Empty if no error. |
| message | string | - | Error message. Empty if no error. |

### Status Values

| Value | Description |
|-------|-------------|
| 0 | STATUS_INVALID |
| 1 | STATUS_PROCESSING |
| 2 | STATUS_DOWNLOADABLE |
| 3 | STATUS_DOWNLOADED |
| 4 | STATUS_FAILED |
