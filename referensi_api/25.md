# Shopee API: v2.payment.get_escrow_detail_batch

## API Description

Use this API to fetch the details of order income by batch.

## Request

### Endpoint
- **Method**: POST
- **Path**: `/api/v2/payment/get_escrow_detail_batch`

### Environment URLs

| Environment | Location | HTTP Address |
|-------------|----------|--------------|
| Live | Global | `https://partner.shopeemobile.com/api/v2/payment/get_escrow_detail_batch` |
| Live | Chinese Mainland | `https://openplatform.shopee.cn/api/v2/payment/get_escrow_detail_batch` |
| Live | Brazil | `https://openplatform.shopee.com.br/api/v2/payment/get_escrow_detail_batch` |
| Sandbox | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/payment/get_escrow_detail_batch` |
| Sandbox | Chinese Mainland | `https://openplatform.sandbox.test-stable.shopee.cn/api/v2/payment/get_escrow_detail_batch` |

### Common Request Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| partner_id | int | 1 | Partner ID assigned upon registration. Required for all requests. |
| timestamp | timestamp | 1610000000 | Timestamp of the request. Required for all requests. Expires in 5 minutes. |
| access_token | string | c09222e3fc40ffb25fc947f738b1abf1 | Token for API access. Valid for multiple use, expires in 4 hours. |
| shop_id | int | 600000 | Shopee's unique identifier for a shop. Required for most APIs. |
| sign | string | e318d3e932719916a9f9ebb57e2011961bd47abfa54a36e040d050d8931596e2 | Signature generated via HMAC-SHA256. |

### Request Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| order_sn_list | string[] | True | Shopee's unique identifiers for orders. Limit: [1, 50]. Recommended: 1–20 orders per request. |

### Request Example

```json
{
    "order_sn_list": [
        "23081814A9CP4E",
        "2308180XYTC5M5",
        "230817V7B3XAMD"
    ]
}
```
