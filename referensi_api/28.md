# Shopee API: v2.payment.generate_income_report

## API Description

Trigger income report generation.

## Request

### Endpoint
- **Method**: GET
- **Path**: `/api/v2/payment/generate_income_report`

### Environment URLs

| Environment | Location | HTTP Address |
|-------------|----------|--------------|
| Live | Global | `https://partner.shopeemobile.com/api/v2/payment/generate_income_report` |
| Live | Chinese Mainland | `https://openplatform.shopee.cn/api/v2/payment/generate_income_report` |
| Live | Brazil | `https://openplatform.shopee.com.br/api/v2/payment/generate_income_report` |
| Sandbox | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/payment/generate_income_report` |
