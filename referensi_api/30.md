# Shopee API: v2.payment.get_income_overview

## API Description

Retrieves a consolidated snapshot of the seller's income amounts categorized by income status for a specified shop. This API provides a holistic overview similar to Seller Center's "Income Overview" section, allowing external systems to reflect the same current payout view.

Data is dynamically determined based on the shop type (Local or Cross Border) and the income status requested. Historical income results are not retrievable, providing consistent information as Seller Centre.

## Request

### Endpoint
- **Method**: GET
- **Path**: `/api/v2/payment/get_income_overview`

### Common Request Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| partner_id | int | 1 | Partner ID assigned upon registration. Required for all requests. |
| timestamp | timestamp | 1610000000 | Timestamp of the request. Required for all requests. Expires in 5 minutes. |
| access_token | string | c09222e3fc40ffb25fc947f738b1abf1 | Token for API access. Valid for multiple use, expires in 4 hours. |
| shop_id | int | 600000 | Shopee's unique identifier for a shop. Required for most APIs. |
| sign | string | e318d3e932719916a9f9ebb57e2011961bd47abfa54a36e040d050d8931596e2 | Signature generated via HMAC-SHA256. |

### Request Parameters

| Name | Type | Required | Sample | Description |
|------|------|----------|--------|-------------|
| income_status | int32 | False | 1 | Status of Seller Income payout. See values below. If not provided, API returns all values for all income statuses. |

### Income Status Values

**Local Shop:**
| Value | Description |
|-------|-------------|
| 1 | Released |
| 2 | Pending |

**CB (Cross Border) Shop:**
| Value | Description |
|-------|-------------|
| 0 | To Release |
| 1 | Released |
| 2 | Pending |

### Request Example

```
GET /api/v2/payment/get_income_overview?access_token=access_token&income_status=1&partner_id=partner_id&shop_id=shop_id&sign=sign&timestamp=timestamp
```

## Response

### Response Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| error | string | - | Error type if hit error. Empty if no error. |
| message | string | - | Error details if hit error. Empty if no error. |
| request_id | string | e3e3e7f34151e704736b612a6b9ae101 | Identifier for an API request for error tracking. |
| response | object | - | Response content. |
| response.latest_payout_date | string | 2025-08-19 | The latest payout date for the released income. Format: YYYY-MM-DD. Only for CN shops. |
| response.total_income | object | - | Object containing total income components. |
| response.total_income.pending_amount | float | 330598.87 | Total amount pending release (Local: orders before ESCROW_PAID; CB: orders before ESCROW_PAYOUT). |
| response.total_income.to_release_amount | float | 330598.87 | Amount queued for release in the next payout cycle (CB only). Not applicable for Local shops. |
| response.total_income.released_amount | float | 330598.87 | Total amount successfully released to the seller. |
