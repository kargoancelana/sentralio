# Shopee API: v2.logistics.get_shipping_document_parameter

## API Description

Use this api to get the selectable shipping_document_type and suggested shipping_document_type.

## Request

### Endpoint
- **Method**: POST
- **Path**: `/api/v2/logistics/get_shipping_document_parameter`

### Environment URLs

| Environment | Location | HTTP Address |
|-------------|----------|--------------|
| Live | Global | `https://partner.shopeemobile.com/api/v2/logistics/get_shipping_document_parameter` |
| Live | Chinese Mainland | `https://openplatform.shopee.cn/api/v2/logistics/get_shipping_document_parameter` |
| Live | Brazil | `https://openplatform.shopee.com.br/api/v2/logistics/get_shipping_document_parameter` |
| Sandbox | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/logistics/get_shipping_document_parameter` |
| Sandbox | Chinese Mainland | `https://openplatform.sandbox.test-stable.shopee.cn/api/v2/logistics/get_shipping_document_parameter` |

| Name | Type | Sample | Description |
|------|------|--------|-------------|

| shop_id | int | 600000 | Shopee's unique identifier for a shop. Required param for most APIs. |

### Request Parameters

| Name | Type | Required | Sample | Description |
|------|------|----------|--------|-------------|
| order_list | object[] | True | - | The list of orders you want to get. limit [1,50] |
| order_list.order_sn | string | True | 201201E81SYYKE | Shopee's unique identifier for an order. |
| order_list.package_number | string | False | 60489687088750 | Shopee's unique identifier for the package under an order. You should't fill the field with empty string when there is't a package number. |
