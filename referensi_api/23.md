# Shopee API: v2.payment.get_payment_method_list

## API Description

Obtain payment method list (no authentication required).

## Request

### Endpoint
- **Method**: GET
- **Path**: `/api/v2/payment/get_payment_method_list`

### Environment URLs

| Environment | Location | HTTP Address |
|-------------|----------|--------------|
| Live | Global | `https://partner.shopeemobile.com/api/v2/payment/get_payment_method_list` |
| Live | Chinese Mainland | `https://openplatform.shopee.cn/api/v2/payment/get_payment_method_list` |
| Live | Brazil | `https://openplatform.shopee.com.br/api/v2/payment/get_payment_method_list` |
| Sandbox | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/payment/get_payment_method_list` |
| Sandbox | Chinese Mainland | `https://openplatform.sandbox.test-stable.shopee.cn/api/v2/payment/get_payment_method_list` |

### Common Request Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| partner_id | int | 1 | Partner ID assigned upon registration. Required for all requests. |
| timestamp | timestamp | 1610000000 | Timestamp of the request. Required for all requests. Expires in 5 minutes. |
| sign | string | e318d3e932719916a9f9ebb57e2011961bd47abfa54a36e040d050d8931596e2 | Signature generated via HMAC-SHA256 using partner_id, api path, timestamp, and partner_key. |

### Request Parameters

No request parameters required.

## Response

### Response Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| error | string | - | Error code. Empty if no error. |
| message | string | - | Error message. Empty if no error. |
| request_id | string | - | Unique identifier for the request. |
| response | object[] | - | List of payment methods grouped by region. |
| response[].payment_method | string[] | - | List of payment method names available in the region. |
| response[].region | string | ID | Region code (e.g., ID, MY, PH, SG, TH, TW, VN). |
