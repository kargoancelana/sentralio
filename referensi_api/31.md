# Shopee API: v2.payment.get_income_detail

## API Description

Retrieves detailed order-level income information across various income statuses for a specified time period. This API enables partners to display granular transaction-level income data consistent with Seller Center's "Income Details" view, segmented by income status and payout stage.

The API dynamically adapts data fields based on the seller's shop type (Local or Cross Border) and the selected income status (e.g., Pending, To Release, Released).

## Request

### Endpoint
- **Method**: GET
- **Path**: `/api/v2/payment/get_income_detail`

|------|------|--------|-------------|
| partner_id | int | 1 | Partner ID assigned upon registration. Required for all requests. |
| timestamp | timestamp | 1610000000 | Timestamp of the request. Required for all requests. Expires in 5 minutes. |
| access_token | string | c09222e3fc40ffb25fc947f738b1abf1 | Token for API access. Valid for multiple use, expires in 4 hours. |

### Request Parameters

| Name | Type | Required | Sample | Description |
|------|------|----------|--------|-------------|
| date_from | string | True | 2025-09-25 | Start date (YYYY-MM-DD) of the income reference period. Only used for `income_status = Released`. Other statuses display all records currently in that status. Date range cannot exceed 14 days. |
| date_to | string | True | 2025-09-30 | End date (YYYY-MM-DD) of the income reference period. Must be later than `date_from`. Only used for `income_status = Released`. Date range cannot exceed 14 days. |
| income_status | int32 | True | 1 | Status of Seller Income payout. See values below. |
| cursor | string | False | 176714986216530 | Pagination token for the next set of results. Use empty string `""` for the first request. |
| page_size | int64 | True | 30 | Number of income detail records to retrieve per page. |

### Income Status Values

**Local Shop:**
| Value | Description |
|-------|-------------|
| 1 | Released |
| 2 | Pending |

**CB (Cross Border) Shop:**
| Value | Description |
|-------|-------------|
| 0 | To Release |
| 1 | Released |
| 2 | Pending |
