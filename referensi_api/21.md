# Shopee API: v2.logistics.get_tracking_number

## API Description

After arranging shipment (`v2.logistics.ship_order`) for the integrated channel, use this api to get the tracking_number, which is a required parameter for creating shipping labels.

**Important Notes:**
- The api response can return tracking_number empty, since this info is dependent from the 3PL.
- Due to this it is allowed to keep calling the api within **5 minutes interval**, until the tracking_number is returned.

## Request

### Endpoint
- **Method**: GET
- **Path**: `/api/v2/logistics/get_tracking_number`

| Name | Type | Sample | Description |
|------|------|--------|-------------|

| shop_id | int | 600000 | Shopee's unique identifier for a shop. Required param for most APIs. |
| sign | string | e318d3e932719916a9f9ebb57e2011961bd47abfa54a36e040d050d8931596e2 | Signature generated by partner_id, api path, timestamp, access_token, shop_id and partner_key via HMAC-SHA256 hashing algorithm. |

### Request Parameters

| Name | Type | Required | Sample | Description |
|------|------|----------|--------|-------------|
| order_sn | string | True | 201214JASXYXY6 | Shopee's unique identifier for an order. |

| response_optional_fields | string | False | first_mile_tracking_number | Indicate response fields you want to get. Available values: `plp_number`, `first_mile_tracking_number`, `last_mile_tracking_number`. Multiple fields separated by comma. |

## Catatan Penting

1. Ini adalah API untuk **single order** — untuk batch gunakan `v2.logistics.get_mass_tracking_number` (file 11.md)
2. Dipanggil **setelah** `v2.logistics.ship_order` (file 19.md)
3. `tracking_number` bisa **kosong** karena bergantung pada 3PL — polling setiap 5 menit sampai tersedia
4. Tracking number **wajib** sebelum bisa `create_shipping_document` (file 13.md)
5. Untuk split order, wajib menyertakan `package_number`

## Flow Lengkap Single Order (READY_TO_SHIP → Cetak Label)

```
1. get_shipping_parameter (file 20.md)  → cek pickup/dropoff/non_integrated
2. ship_order (file 19.md)              → atur pengiriman
3. get_tracking_number (file 21.md)     → polling sampai tracking number tersedia
4. create_shipping_document (file 13.md) → buat task label
5. get_shipping_document_result (file 14.md) → poll sampai READY
6. download_shipping_document (file 15.md)   → download PDF label
```
