# Shopee API: v2.payment.generate_income_statement

## API Description

Trigger income statement generation.

## Request

### Endpoint
- **Method**: GET
- **Path**: `/api/v2/payment/generate_income_statement`

### Environment URLs

| Environment | Location | HTTP Address |
|-------------|----------|--------------|
| Live | Global | `https://partner.shopeemobile.com/api/v2/payment/generate_income_statement` |
| Live | Chinese Mainland | `https://openplatform.shopee.cn/api/v2/payment/generate_income_statement` |
| Live | Brazil | `https://openplatform.shopee.com.br/api/v2/payment/generate_income_statement` |
| Sandbox | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/payment/generate_income_statement` |
| Sandbox | Chinese Mainland | `https://openplatform.sandbox.test-stable.shopee.cn/api/v2/payment/generate_income_statement` |

### Common Request Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| partner_id | int | 1 | Partner ID assigned upon registration. Required for all requests. |
| timestamp | timestamp | 1610000000 | Timestamp of the request. Required for all requests. Expires in 5 minutes. |
| access_token | string | c09222e3fc40ffb25fc947f738b1abf1 | Token for API access. Valid for multiple use, expires in 4 hours. |
| shop_id | int | 600000 | Shopee's unique identifier for a shop. Required for most APIs. |
| sign | string | e318d3e932719916a9f9ebb57e2011961bd47abfa54a36e040d050d8931596e2 | Signature generated via HMAC-SHA256. |

### Request Parameters

| Name | Type | Required | Sample | Description |
|------|------|----------|--------|-------------|
| release_time_from | int64 | True | 1751302800 | Start time. Must be Monday (local time) for weekly report, or 1st day of month for monthly report. |
| release_time_to | int64 | True | 1753981199 | End time. Must be Sunday (local time) for weekly report, or last day of month for monthly report. |
| statement_type | int32 | True | 1 | Statement type: `1` = WEEKLY, `2` = MONTHLY. Required for local sellers. CB sellers do not require this. |

### Request Example

```
GET /api/v2/payment/generate_income_statement?access_token=access_token&partner_id=partner_id&release_time_from=1751302800&release_time_to=1753981199&shop_id=shop_id&sign=sign&statement_type=1&timestamp=timestamp
```

## Response

### Response Parameters

| Name | Type | Sample | Description |
|------|------|--------|-------------|
| response | object | - | Response content. |
| response.id | int64 | 123456789 | Identifier of the generated income statement file. |
| error | string | - | Error code. Empty if no error. |
