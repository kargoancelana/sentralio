# Shopee API: v2.logistics.get_mass_tracking_number

## API Description

After arranging shipment (`v2.logistics.mass_ship_order`) for the integrated channel, use this api to get the tracking_number, which is a required parameter for creating shipping labels. 

**Important Notes:**
- The api response can return tracking_number empty, since this info is dependent from the 3PL.
- Due to this it is allowed to keep calling the api within 5 minutes interval, until the tracking_number is returned.

## Request

### Endpoint
- **Method**: POST
- **Path**: `/api/v2/logistics/get_mass_tracking_number`

| Name | Type | Required | Sample | Description |

| package_list | object[] | True | - | The list of packages you want to get tracking number. limit [1, 50]. |

| response_optional_fields | string | False | first_mile_tracking_number | Indicate response fields you want to get. Please select from the below response parameters. If you input an object field, all the params under it will be included automatically in the response.
