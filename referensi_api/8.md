# Shopee API: v2.logistics.get_mass_shipping_parameter

## API Description

Use this API to check if package support pickup, dropoff, non-integrated. For pickup, return address and pickup time id options. For dropoff, return branch id, sender real name, etc. Can batch request for packages under same product_location_id and logistics_channel_id.

**When to call this API:**
- Fulfillment status is LOGISTICS_READY; or
- Fulfillment status is LOGISTICS_PICKUP_RETRY; or
- Fulfillment status is LOGISTICS_REQUEST_CREATED and meet Instant Order Reschedule conditions

## Request

### Endpoint
- **Method**: POST
- **Path**: `/api/v2/logistics/get_mass_shipping_parameter`
