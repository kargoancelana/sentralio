# Shopee API: v2.payment.get_escrow_list

## API Description

Use this API to fetch the accounting list of orders.

## Request

### Endpoint
- **Method**: GET
- **Path**: `/api/v2/payment/get_escrow_list`

### Environment URLs

| Environment | Location | HTTP Address |
|-------------|----------|--------------|
| Live | Global | `https://partner.shopeemobile.com/api/v2/payment/get_escrow_list` |
| Live | Chinese Mainland | `https://openplatform.shopee.cn/api/v2/payment/get_escrow_list` |
| Live | Brazil | `https://openplatform.shopee.com.br/api/v2/payment/get_escrow_list` |
| Sandbox | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/payment/get_escrow_list` |
| Sandbox | Chinese Mainland | `https://openplatform.sandbox.test-stable.shopee.cn/api/v2/payment/get_escrow_list` |
