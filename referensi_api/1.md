# Shopee Order API - v2.order.get_order_list

## Overview
API ini digunakan untuk mencari dan mengambil daftar order dari Shopee. Anda dapat memfilter order berdasarkan status jika diperlukan.

## Endpoint Details

### HTTP Method
`GET`

### Request Address

| Environment | Location | HTTP Address |
|------------|----------|--------------|
| **Live** | Global | `https://partner.shopeemobile.com/api/v2/order/get_order_list` |
| | Chinese Mainland | `https://openplatform.shopee.cn/api/v2/order/get_order_list` |
| | Brazil | `https://openplatform.shopee.com.br/api/v2/order/get_order_list` |
| **Sandbox** | Global | `https://openplatform.sandbox.test-stable.shopee.sg/api/v2/order/get_order_list` |

| Name | Type | Required | Sample | Description |

| `cursor` | string | False | `""` | Starting entry untuk pagination. Default: empty |

## Implementation Notes

### Pagination
1. Set `page_size` (1-100) untuk mengontrol jumlah entries per page
2. Jika `response.more` = `true`, ada data lebih lanjut
3. Gunakan `response.next_cursor` sebagai `cursor` di request berikutnya
4. Loop sampai `response.more` = `false`

### Time Range
- Maximum range: **15 hari**
- Field yang bisa digunakan: `create_time` atau `update_time`
- Timestamp dalam format Unix timestamp

### Authentication
- Signature menggunakan HMAC-SHA256
- Access token expires dalam 4 jam
- Request expires dalam 5 menit (berdasarkan timestamp)
