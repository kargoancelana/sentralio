# Shopee API: v2.logistics.get_shipping_parameter

## API Description

Use this api to check if the package support pickup, dropoff, or non-integrated method. For pickup, will return addresses and pickup time id options. For dropoff, will return branch id, sender real name etc, depending on 3PL requirements.

**When to call this API:**
1. Fulfillment status is `LOGISTICS_READY`; or
2. Fulfillment status is `LOGISTICS_PICKUP_RETRY`; or
3. Fulfillment status is `LOGISTICS_REQUEST_CREATED` and meet Instant Order Reschedule Pickup conditions

## Catatan Penting

1. Ini adalah API untuk **single order** — untuk batch gunakan `v2.logistics.get_mass_shipping_parameter` (file 8.md)
2. Harus dipanggil **sebelum** `v2.logistics.ship_order` (file 19.md)
3. Gunakan `info_needed` untuk menentukan metode mana yang tersedia (pickup/dropoff/non_integrated)
4. Untuk split order, wajib menyertakan `package_number`
5. Pilih `address_id` dan `pickup_time_id` dari response untuk digunakan di `ship_order`
6. Untuk dropoff, pilih `branch_id` dari `branch_list`
7. `time_slot_list` bisa null untuk beberapa channel — dalam kasus ini, Shopee akan arrange timing secara otomatis
