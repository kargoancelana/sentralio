# Shopee API: v2.logistics.mass_ship_order

## API Description

Use this API to initiate logistics including arrange pickup, dropoff or shipment for non-integrated logistic channels. Should call `v2.logistics.get_mass_shipping_parameter` to fetch all required params first. 

**Important Notes:**
- It's recommended to initiate logistics one hour after the orders were placed since there is one-hour window buyer can cancel any order without request to seller.
- The API can only batch arrange shipment for multiple packages under the same product_location_id and same logistics_channel_id.

## Request

### Endpoint
- **Method**: POST
- **Path**: `/api/v2/logistics/mass_ship_order`
